Detailed Description

The set of config stored for each distinct (as determined by Component::payload_type(), essentially C++ built-in std::type_info) component payload type (in English – component enum class).

As I write this, there is only one member, so we did not need a struct, but it really felt like it might get extended later – and there is zero performance loss from this, and the extra code is minimal.
