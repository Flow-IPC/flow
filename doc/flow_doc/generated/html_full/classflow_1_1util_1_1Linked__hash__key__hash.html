Detailed Description

template<typename Hash>
class flow::util::Linked_hash_key_hash< Hash >

The internal-use Hash functor wrapper, used as the hasher type in internal-use set-type Linked_hash_key_set.

Impl note: We store the Hash via private inheritance, making use of Empty Base-class Optimization (EBO) if possible; namely when Hash is an empty type (which is typical); in which case this shall waste no space.
