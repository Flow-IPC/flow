Detailed Description

A data store that keeps stats about the outgoing direction of a Peer_socket connection to another Flow-protocol Peer_socket.

Note that this also includes stats about incoming traffic that facilitates further outgoing traffic (namely, data on incoming acknowledgments are in this class).

All other comments in Peer_socket_receive_stats doc header apply equally to this class.
