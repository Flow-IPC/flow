Detailed Description

Empty struct suitable as a *_value_set template arg for Config_manager, when a slot requires a Value_set, but you have no config to actually parse there.

In particular, if you have a static Value_set but no corresponding dynamic one, you should provide this as the template arg.

See also
null_declare_opts_func() and null_final_validator_func().
