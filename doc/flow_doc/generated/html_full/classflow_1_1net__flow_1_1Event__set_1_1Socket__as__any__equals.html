Detailed Description

Equality predicate class used in storing various sockets of types wrapped as boost::anys in the Sockets type.

We do not simply define operator==(any, any), because our implementation specifically works for any objects that store various sockets and won't work for all other possibilities; and we don't want to force this on every user of hash<any>.
